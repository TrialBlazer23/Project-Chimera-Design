# Contributing to Project Chimera

Thank you for your interest in contributing to the Neuro-Symbolic Agent Protocol (NSAP) specification.

## Ways to Contribute

### 1. RFC Submissions

Propose new specifications or amendments through the RFC (Request for Comments) process:

1. **Fork** this repository
2. **Create** a new RFC document in `RFCs/Proposed/` using the template below
3. **Submit** a Pull Request for community review

### 2. Specification Improvements

- Clarify existing language
- Fix inconsistencies
- Add examples
- Improve documentation

### 3. Reference Implementations

- Submit implementations in `Implementations/Examples/`
- Follow the SDR type specifications exactly
- Include comprehensive test cases

## RFC Template

```markdown
# RFC-NNNN: [Title]

**Status:** Proposed
**Created:** YYYY-MM-DD
**Authors:** [Names]

## Abstract

Brief summary of the proposal.

## Motivation

Why is this change needed?

## Specification

Technical details of the proposal.

## Backwards Compatibility

Impact on existing implementations.

## Security Considerations

Safety and security implications.

## References

Related specifications and prior art.
```

## Code of Conduct

All contributors must adhere to our community standards of respectful, constructive collaboration.

## Questions?

Open an issue for discussion.
